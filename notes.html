Can I use props for variables? Yes, props are used kind of like variables but with a different syntax, for example you can use the code below and just get 3 different buttons, each one with a different label. 
<button-primary {label}="button-1">...</button-primary>
<button-primary {label}="button-2">...</button-primary>
<button-primary {label}="button-3">...</button-primary>
